## 📖 Descrição
Este projeto tem como objetivo desenvolver um sistema em Node.js capaz de ler um arquivo Markdown e extrair referências (links) utilizando Expressões Regulares (RegEx).

O sistema processa o arquivo `Projeto2_arquivo.md` e exibe no console o nome da referência e sua URL de forma estruturada.

---

## 🚀 Tecnologias utilizadas
- Node.js
- Módulo `fs/promises`
- Expressões Regulares (RegEx)
- Biblioteca `chalk`
